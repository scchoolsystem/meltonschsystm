📄 Data Processing Agreement (DPA)

Roles and Responsibilities

SmartDev acts as a Data Processor on behalf of educational institutions (Data Controllers). We process data strictly according to institutional instructions.

Data Processing Details

  • Student Data: Enrollments, grades, attendance, behavior records
  • Staff Data: Employment info, payroll, performance records
  • Administrative Data: System configurations, audit logs, access records

Subprocessors

We use certified cloud providers (AWS, Cloudflare, Supabase) for infrastructure. All subprocessors comply with data protection standards.

Security Measures

  • Encryption at rest (AES-256) and in transit (TLS 1.3)
  • Regular security audits and penetration testing
  • Access controls and authentication mechanisms
  • Automated backup and disaster recovery

Incident Response

Security incidents are investigated immediately. Affected institutions are notified within 72 hours of confirmation.

Data Deletion

Upon contract termination, all institutional data is permanently deleted within 90 days.
